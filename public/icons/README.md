# PWA Icons Needed

This directory needs the following icon files for the PWA manifest:

- `icon-192.png` — 192×192px app icon
- `icon-512.png` — 512×512px app icon

## Design Requirements

Icons should follow the Where2Meet.Me design system:
- Primary color: Deep Navy `#000666`
- Background: Light surface `#f9f9fb`
- Consider using the W2M logo or a location pin motif
- Icons should work both with and without masking (safe zone: 80% of canvas)
